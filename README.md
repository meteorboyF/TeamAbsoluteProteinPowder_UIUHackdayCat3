## üì¶ Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/meteorboyF/TeamAbsoluteProteinPowder_UIUHackdayCat3.git
   cd TeamAbsoluteProteinPowder_UIUHackdayCat3
   ```

2. **Install Dependencies**
   ```bash
   composer install
   npm install
   ```

3. **Configure Environment**
   - Copy `.env.example` to `.env`.
   - Set `DB_CONNECTION=mongodb` and configure your MongoDB credentials.
   - Add your `GEMINI_API_KEY`.
   - Run `php artisan key:generate`.

4. **Run the App**
   ```bash
   npm run dev
   php artisan serve
   ```
